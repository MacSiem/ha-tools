8. **ACCESSIBILITY**  
- Color contrast:  
  - Status badge colors (#10B981 green, #EF4444 red, #94A3B8 gray) on white background likely pass contrast (green and red are bright).  
  - Text colors mostly use Bento tokens which adapt to dark mode.  
  - Inline text colors in network details use `var(--ts)` (secondary text) which might be low contrast but acceptable for secondary info.  
- Missing ARIA labels:  
  - Tab buttons have no explicit `aria-selected` or `role="tab"` attributes.  
  - Pagination buttons have no aria-labels (e.g. "Previous page", "Next page").  
  - Search input has placeholder but no label or aria-label.  
  - Filter select and sorting selects have no labels or aria-labels (only visual labels).  
  - Alert dismiss buttons have no aria-label or title attribute; only text "Dismiss".  
- Keyboard focus styles:  
  - Inputs and selects have focus border and box-shadow (line ~410).  
  - Buttons have hover styles but no explicit focus styles visible in CSS.  
**Result:** Some accessibility issues: missing ARIA roles/labels on interactive elements (tabs, pagination, dismiss buttons), and missing explicit labels for inputs/selects. Color contrast generally acceptable.
