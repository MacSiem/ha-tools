**8. ACCESSIBILITY**

- Color contrast:
  - Uses Bento tokens which are designed for good contrast.
  - Hardcoded colors like `#EF4444` (red) and `#10B981` (green) are bright and likely sufficient contrast.
  - Some text uses muted colors like `var(--bento-text-secondary)` and `var(--bento-text-muted)` which may be borderline but typical for secondary text.
- Aria labels:
  - Buttons for removing entities/devices from queue have `aria-label="Remove"` (line ~280).
  - Add single entity buttons do not have explicit aria-label but have visible text "+ Queue".
  - Tab buttons have no aria roles or aria-selected attributes.
  - No keyboard focus styles beyond default.
- Missing:
  - Tabs lack ARIA roles (`tablist`, `tab`, `tabpanel`).
  - No keyboard navigation hints for tabs.
  - No aria-labels on tab buttons.
- **Verdict:** Basic accessibility present, but ARIA roles and keyboard navigation support for tabs missing.
